# Projects
*Powered by EruditeWBT*

This folder is where shipped projects get cataloged and linked (even if the real code lives in separate repos).

## Add a Project
1. Create a file: `projects/<slug>.md`
2. Include: problem, user, architecture diagram, demo link, repo link

Use:
- Pitch template: `templates/PROJECT_PITCH_TEMPLATE.md`
- Review template: `templates/PROJECT_REVIEW_TEMPLATE.md`
