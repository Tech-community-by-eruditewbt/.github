## Trunk (Everyone Starts Here)
Foundations:
- Programming fundamentals
- Systems thinking
- Linux & development environment
- Git & collaboration
- APIs and data flow
- Problem decomposition
- Technical documentation

Minimum standard:
- You can ship small projects end-to-end
- You can explain your system in diagrams + README
- You can debug with logs, not vibes
