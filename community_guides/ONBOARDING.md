# Onboarding (New Member Start Here)
*Powered by EruditeWBT*

## The One-Sentence Mission
We turn **curiosity into engineering power and income** through structured building.

## Your First 60 Minutes
1. Read: `manifesto/BUILDER_MANIFESTO.md`
2. Read: `tree_v0/ENGINEERING_TREE_V0.md`
3. Choose one starter project: `operations/STARTER_PROJECTS.md`
4. Introduce yourself using the template in `templates/INTRO_POST_TEMPLATE.md`

## Your First 7 Days (Minimum Standard)
By Day 7, you should have:
1. A repo or project folder
2. A README with:
   - problem statement
   - architecture diagram
   - setup steps
   - weekly goal
3. A first demo (even if rough): screenshot/video/GIF

## How We Work Here
- Weekly shipping (small is fine, zero is not)
- Document what you learn
- Ask for reviews early
- Teach forward when you can
