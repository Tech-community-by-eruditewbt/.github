# FAQ
*Powered by EruditeWBT*

## Is this a course?
No. It’s a build-and-review community. Learning happens inside projects.

## Do I need to be “good at coding” first?
No. You need consistency. Start with the Trunk and one starter project.

## What if I’m busy?
Ship small. Weekly progress can be 30–90 minutes if focused.

## How do I get help?
Post:
- what you tried
- logs/errors
- screenshot/video
- your next hypothesis

## Can I monetize immediately?
You can start small (micro freelance tasks), but we optimize for durable skill and proof.
