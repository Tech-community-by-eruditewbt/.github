# Roles (Clear Identity Ladders)
*Powered by EruditeWBT*

## Member Roles
- **Explorer**
  - New member learning the trunk
  - Requirement: choose a starter project + post a plan
- **Apprentice**
  - Building with guidance
  - Requirement: weekly progress + documentation
- **Builder**
  - Shipping projects end-to-end
  - Requirement: 1 shipped project + demo + write-up
- **Engineer**
  - Consistent production-level execution
  - Requirement: 2+ shipped projects + can review others
- **Architect**
  - System designer, mentors builders
  - Requirement: leads a sprint + designs a reusable system
- **Founder**
  - Building products/startups from community systems
  - Requirement: runs a product track or incubator project

## Staff / Operator Roles
- **Mentor**
  - Runs reviews, unblock sessions, office hours
- **Moderator**
  - Protects culture + enforces rules
- **Sprint Lead**
  - Sets weekly goals and runs demos
