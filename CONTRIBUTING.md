# Contributing
*Powered by EruditeWBT*

This community grows through builders contributing knowledge and projects.

## Ways To Contribute
- Improve documentation
- Submit project examples
- Propose new branches or trunk modules
- Mentor new members
- Review projects (actionable feedback)

## Contribution Standards
- Real engineering principles
- Practical projects with a real user (even if it’s you)
- Clear documentation (README + diagram + setup steps)

## How To Submit
- Docs: open a PR that edits markdown files
- Projects: add a project entry under `projects/` and link to the repo/demo
- Builders: add your profile under `builders/`

All contributions must follow `community_guides/CODE_OF_CONDUCT.md`.
