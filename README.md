# Community Launch Pack v1 (EruditeWBT)

Deploy-ready docs you can publish immediately (website/GitHub/Notion/Discord pins).

## What’s Inside
- `manifesto/BUILDER_MANIFESTO.md`
- `tree_v0/ENGINEERING_TREE_V0.md` + diagrams (`.mmd` + `.png`)
- `community_guides/` (onboarding, rules, roles, code of conduct, governance, FAQ)
- `operations/` (90-day roadmap, metrics, rituals, templates, next videos)
- `templates/` (Discord pins, announcements, project pitch/review templates)
- `builders/` (public hall of builders)
- `projects/` (project catalog + templates)

## Fast Deploy (Recommended Order)
1. Create a GitHub repo named `engineering-tree-community`.
2. Upload this folder’s contents (or copy `README_GITHUB.md` to the repo root as `README.md`).
3. Publish:
   - `manifesto/BUILDER_MANIFESTO.md` (the “Why”)
   - `tree_v0/ENGINEERING_TREE_V0.md` + `tree_v0/diagrams/` (the “How”)
4. Launch Discord using:
   - `templates/DISCORD_CHANNELS.md`
   - `operations/DISCORD_PINS_AND_ANNOUNCEMENTS.md`
5. Start Sprint 0 using `operations/WEEKLY_SPRINT.md` and `operations/STARTER_PROJECTS.md`.
