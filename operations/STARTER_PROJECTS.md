# First 5 Starter Projects (v0)
*Powered by EruditeWBT*

## Project 1: Personal Developer Dashboard
Goal: one place that shows your tasks, learning log, and shipped demos.

Minimum features:
- task list + status
- weekly goals
- links to repos/demos
- “what I learned” log

## Project 2: IoT Sensor Monitor
Goal: simulate or read real sensor data and visualize it.

Minimum features:
- telemetry ingestion (real or simulated)
- dashboard chart
- alert when threshold is exceeded

## Project 3: AI Automation Tool (Email Summarizer)
Goal: automate a boring workflow with AI + routing.

Minimum features:
- input → summarize → categorize
- store results (file/db)
- simple UI or CLI

## Project 4: Energy Monitoring System
Goal: show energy data and actionable insights.

Minimum features:
- metrics collection (simulated OK)
- dashboard + daily summary
- “anomaly” or alert rule

## Project 5: Micro SaaS Tool
Goal: the simplest paid tool you can ship.

Minimum features:
- landing page
- one core feature
- basic auth
- deploy
