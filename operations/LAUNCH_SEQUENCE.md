## Step 1: Public Base (GitHub = Source of Truth)
- Publish this repo as `engineering-tree-community`.
- Keep docs here even if you mirror them to Notion/GitBook.

## Step 2: Knowledge Hub (GitBook/Notion)
Mirror these pages first:
- Manifesto
- Engineering Tree v0 (+ diagrams)
- Rules + Onboarding + Roles
- Weekly sprint template

## Step 3: Community Server (Discord)
Create channels from `templates/DISCORD_CHANNELS.md`.

## Step 4: First Pins (Make It Feel “Alive”)
Pin:
- welcome/start-here message
- manifesto link
- rules
- onboarding checklist
- sprint of the week

## Step 5: First Announcement (Launch)
Use: `operations/DISCORD_PINS_AND_ANNOUNCEMENTS.md`

## Step 6: First Week Plan
Day 1: publish intro video + manifesto + tree v0  
Day 2: publish “Why Most People Fail Learning Tech”  
Day 3: start the first challenge + starter project picks  
Day 4: live session (tree explanation)  
Day 5: project pitches  
Day 6: reviews  
Day 7: progress update + highlights
