# 90-Day Roadmap (v0)
*Powered by EruditeWBT*

## Phase 1 (Days 1–14): Foundation + Culture Lock
- Publish manifesto + tree v0
- Establish rules + roles + review culture
- Run Sprint 0: “Ship Something Small”
- Collect 10 starter project repos

Deliverables:
- 10 demos
- 10 READMEs with diagrams
- first “wins” showcase

## Phase 2 (Days 15–45): Tree Activation (Trunk → Branches)
- Trunk workshops (1–2 per week)
- Branch onboarding sessions (software/embedded/energy/data)
- First “Proof Week”: everyone posts progress + blockers

Deliverables:
- 25 active builders
- 4 branch channels + one starter project each

## Phase 3 (Days 46–75): Monetization + Execution Cadence
- Monetization ladder workshop
- “Service Sprint”: build something useful for a real person/business
- Reviews become routine (weekly)

Deliverables:
- 10 portfolio-grade projects
- 5 paid micro-gigs (even small)

## Phase 4 (Days 76–90): Showcase + Incubation
- Demo Day (recorded)
- Select 1–3 incubator candidates
- Publish a public “State of the Community” report

Deliverables:
- Demo Day recording
- incubator shortlist
- next 90-day plan
