# Discord Weekly Cycle (Guided Mode)
*Powered by EruditeWBT*

Goal: turn “interesting paths” into weekly shipped proof.

## Weekly Loop
1. Members generate a path in the graph: `docs/graph.html`
2. Click **Start 7‑Day Sprint**
3. Post daily/weekly updates using:
   - “Copy Discord Update” button in the graph UI, or
   - `templates/DISCORD_SPRINT_UPDATE_TEMPLATE.md`

## Suggested Discord Channel Rhythm
- `#weekly-sprint`: Monday kickoff + sprint pairing
- `#build-in-public`: daily progress + screenshots
- `#project-showcase`: end-of-week proof posts

## What To Track (Light)
- Day 1: problem statement link
- Day 5: diagram + README link
- Day 7: published proof link
